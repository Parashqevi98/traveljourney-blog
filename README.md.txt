# TravelJourney Blog

TravelJourney is a fullstack travel blog platform developed as an academic and portfolio project.  
The platform allows users to explore travel posts and share tourist experiences.

## Tech Stack

### Backend
- C# / .NET
- RESTful API
- SQL Server

### Frontend
- HTML
- CSS
- JavaScript

## Project Structure

- backend/BlogAPI – Backend API and business logic  
- frontend/TravelJourney – Frontend user interface  

## Notes
This repository contains only demo content.  
User-uploaded images and sensitive configuration files are excluded.
